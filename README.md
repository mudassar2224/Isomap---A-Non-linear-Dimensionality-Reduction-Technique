# 📊 Human Action Recognition using Isomap

## 🔥 Project Overview

This project explores **nonlinear dimensionality reduction** using Isomap to visualize human activity image data.

We transform high-dimensional image features into a 2D/3D space to uncover hidden patterns and similarities between different human actions.

---

## 🎯 Objective

* Understand complex image data using **manifold learning**
* Visualize clusters of human activities
* Analyze similarities between actions like running, sitting, sleeping, etc.

---

## 🧠 Techniques Used

* Image preprocessing (OpenCV)
* Feature extraction (grayscale + flattening)
* Standardization (StandardScaler)
* PCA (noise reduction)
* Isomap (nonlinear dimensionality reduction)
* Silhouette Score (cluster evaluation)
* Plotly (interactive visualization)

---

## ⚙️ Workflow Pipeline

1. Dataset Download (Kaggle)
2. Data Audit (visual inspection)
3. Feature Extraction (image → vector)
4. Data Scaling
5. PCA (dimensionality reduction)
6. Isomap (manifold learning)
7. Visualization (interactive scatter plot)
8. Cluster Evaluation (Silhouette Score)

---

## 📊 Results

* Successfully visualized high-dimensional data in 2D space
* Observed clustering of similar activities
* Identified overlaps between similar actions (e.g., sitting vs lying)

---

## 🧪 Evaluation Metric

**Silhouette Score:** Measures how well clusters are separated.

* Score closer to +1 → better clustering
* Score near 0 → overlapping clusters
* Score negative → poor clustering

---

## 📁 Dataset

Dataset used:
Human Action Recognition (HAR) Dataset from Kaggle

Contains:

* Multiple human activities (running, sitting, sleeping, etc.)
* Image data with labels

---

## 💡 Key Insights

* Isomap captures **nonlinear relationships** better than linear methods
* Some activities naturally overlap due to visual similarity
* High-dimensional image data can be effectively visualized using manifold learning

---
